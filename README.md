# Concurso Soy Loqueleo 2026

Landing del concurso con fondo acuarela espacial (`fondo.png`) y diseño basado en el mockup de referencia.

## Stack

Astro 5 · React (countdown + filtro) · Tailwind CSS 4

## Assets

- `public/fondo.png` — fondo largo que se desplaza con el scroll
- `public/loqueleo.png` — logo en el header

## Desarrollo

```bash
npm install
npm run dev
```
